# Telegram Product Availability Tracker

A Python-based Telegram bot that monitors smartphone product pages and sends real-time notifications when products become available for purchase.

## Features

* Tracks multiple smartphone product pages
* Detects stock availability automatically
* Sends instant Telegram alerts
* Configurable checking interval
* Lightweight implementation using Python and Requests

## Tech Stack

* Python
* Requests
* Telegram Bot API
* Web Scraping

## How It Works

The bot periodically checks Motorola product pages and analyzes stock-status indicators. When a product becomes available, a Telegram notification is sent automatically.

## Example Use Case

* Track newly launched smartphones
* Monitor flash-sale products
* Get notified without manually refreshing product pages

## Future Improvements

* Support for additional e-commerce websites
* Product selection through Telegram commands
